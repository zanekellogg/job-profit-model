This demo gives you a firsthand look at how business intelligence can enhance the way you track profitability and compare estimates to actual revenue. But the real value comes when you see your own data in action.

By uploading your own data, you’ll get instant insights tailored to your business—no generic examples, just real numbers that matter to you. Discover trends, spot opportunities, and see what’s possible with a custom dashboard built specifically for your business.

Follow the steps below to upload your data and start exploring!

To use your own data in the dashboard, prepare a CSV file with the following columns:

  • Job Date: An ISO date string (e.g., 2024-10-01)
  • Description: A reference description for the job
  • Labor: Dollar amount spent on labor
  • Materials: Dollar amount spent on materials
  • Overhead: Dollar amount spent on overhead
  • Amount Received: The amount you were paid for the job
  • Original Estimate: The estimated cost of the job

If you're not familiar with CSV files, here are a few helpful tips:

  • Using Excel: Create your data in Excel and then select File > Save As and choose CSV (Comma delimited) (*.csv) as the file type.
  • Using a Text Editor: You can create a plain text file where each line represents a row and each value is separated by a comma. For example:
    2024-10-01,"Job description",500,300,150,1000,1200
  • Do not add an extra spaces between columns.

Need more detailed help? Check out this guide from Microsoft or search for "how to create a CSV file" online.
